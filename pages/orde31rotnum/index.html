ぐるぐる数 2019.3.9

問題の概要

ある数を b 進数で表現する。
このとき、左端を除くすべての桁が以下の条件を満たす数のことを「 ぐるぐる数 」と呼ぶ：

  • 左隣の桁と等しい
  • ( 左隣の桁の数字 +1 ) % b と等しい ( ※ “%” は剰余の計算 )

例を挙げる：

b 数(b進数表記) ぐるぐる数？
10 6 TRUE
2 1101010100 TRUE
3 12222 TRUE
4 11223000 TRUE
10 234556677 TRUE
16 455 TRUE
36 gh TRUE
10 14 false
3 12010 false
4 30013011 false
10 567788993 false
16 167 false
36 6j false

b と、2つの数 x, y を与える。
x以上で y以下の「 b 進数の ぐるぐる数 」が何個あるのか計算せよ。

入力

入力はこんな感じ：
4,1313,3012
b, x, y がコンマ区切りで並んでいる。
b は 十進数。
x と y は b 進数になっている。

出力

出力は x 以上 y 以下の「b 進数の ぐるぐる数 」の個数を 十進数 で。

上記の例だと
  2222, 2223, 2230, 2233, 2300, 2301, 2330, 2333, 3000, 3001, 3011, 3012
の 12 個の「 4 進数の ぐるぐる数 」があるので
12
を出力すればよい。

補足

  • 1≦x≦y である。
  • y の上限は、まあサンプルデータにあるぐらいの感じで。
  • 2≦b≦36 である。
  • 実装ができた方は Qiitaの記事 のコメント欄からリンクを張っていただくと見つけやすくて助かります。
  • この問題は2019年3月9日に開催される オフラインリアルタイムどう書くE31 の問題です。イベントについては Doorkeeper のページ の方を御覧下さい。
